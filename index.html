HTML Tags and Attributes Reference with CSS Styling!

You can find here information about some HTML tags and attributes combined with CSS styles.

What are the HTML Tags?

An HTML tag is a fundamental component of HTML (Hypertext Markup Language) , the language used to create web pages. Tags are enclosed in angle brackets < > and typically come in pairs: an opening tag and a closing tag. The opening tag denotes the beginning of an HTML element, and the closing tag denotes the end of that element. For example, <p> is the opening tag for a paragraph element, and </p> is the closing tag.

What are the HTML Elements?

An HTML element consists of an opening tag, content, and a closing tag. The content is the information or text contained within the element. Elements can also have attributes, which provide additional information about the element or specify its behavior. For instance, in the HTML element <a href="https://www.example.com"> Click here </a>, so the <a> is the opening tag, Click here is the content, and </a> is the closing tag. The href attribute specifies the URL to which the link leads.

No. Keyword "Tag" Elements Explanation
1 Anchor Tag <a></a> This is the element for using linking pages.
2 Head Tag <head></head> Defines HTML headings from the largest (h1) to the smallest (h6).
3 Body Tag <body></body> This is the body tage which contain the whole contents of an HTML page.
4 Paragraph Tag <p></p> Defines a paragraph.
5 Image Tag <img></img> Embeds an image into the HTML page. The src attribute specifies the path to the image, and the alt attribute provides alternative text.
6 Table Tag <table></table> Defines a table. The <tr> tag defines a row, and the <td> tag defines a cell.
7 Form Tag <form></form> Defines an HTML form for user input. The action attribute specifies where to send the form data, and the method attribute specifies how to send it.
8 Input Tag <input></input> Defines an input control. The type attribute specifies the type of input, such as text, checkbox, or radio button.
9 Main Tag <main></main> The main tag in HTML is used to enclose the primary content of a web page, representing the main purpose or focus of the document.
10 Div Tag <div></div> Defines a division or a section in an HTML document. It is often used with CSS to style and position content.
